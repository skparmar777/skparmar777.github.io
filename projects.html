Facetunes | CS 196

Urbana-Champaign, IL

For CS 196 (Intro to CS Freshman Honors ), I worked in a team of six to develop an android app that uses facial recognition to autogenerate musical playlists based on sentiment analysis using the Spotify Android API. Later on, we added a feature that allows the user to choose an emoji to represent their mood and generate a playlist based on it. I worked on both the front end and backend. For the frontend, I helped design the overall app layout and used Android Studio to create the Camera manifest. For the backend, I worked on integration with the Spotify Android SDK to the emoji and facial values in order to generate playlists.
